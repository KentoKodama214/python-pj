## 世界の総人口の推移
まずはじめに、1950年〜2100年の全世界の国別の人口の推移を調べた。その結果からわかることは、2021年以降は
- <font color="Blue">日本の人口は伸びていない</font>
- インドの著しい伸びで減少傾向の中国を追い越す
- アジア諸国では、インド・中国を除くとインドネシア・パキスタンで人口が伸びている
- ヨーロッパ諸国の人口は伸びておらず、ほぼ横ばい
- 北米〜南米では、<font color="Red">アメリカの人口は2100年まで伸び続けている</font>ものの、その他はほぼ横ばい
- オーストラリアもほぼ横ばい
- <font color="Red">アフリカ諸国では、ナイジェリアを中心にエジプト・エチオピア・コンゴ民主共和国など多くの国で人口が伸びている</font>

## 2021年時点の総人口の上位20カ国の人口密度の推移
2021年時点の総人口の上位20カ国に対して、1950年〜2100年の人口密度の推移をグラフ化した。その結果からわかることは、
- 日本は2021年時点では第3位の人口密度だが、今後は人口減少に伴い人口密度も低下する
- バングラデシュが圧倒的に人口密度が高い
- 人口の急激な増加に伴い、ナイジェリアの人口密度が今後急激に増加する
- エジプト・エチオピア・コンゴ民主共和国などのアフリカ諸国や、インド・ベトナム・フィリピン・インドネシアなどのアジア諸国で人口密度が増加傾向にある
- 中国は広大な国土のため人口密度はそれほど高くないが、インドは高い

## 2021年時点でGDPの上位20カ国の2027年までのGDPの推移
2020年時点のNGDPD(※1)とPPPSH(※2)の上位20カ国に対して、1980年〜2026年のNGDPD/PPPSHの推移をグラフ化した。その結果からわかることは、
- <font color="Red">中国がNGDPD・PPPSHともに第1位(※3)で圧倒的に高く、著しい上昇傾向にある</font>
- 日本は2020年からNGDPDがわずかに上昇傾向にあるものの、PPPSHは1991年のバブル崩壊から減少傾向にある
- インドはNGDPDがわずかに上昇傾向で、PPPSHは2009年から日本を抜いて第2位(※3)で今後も上昇傾向にある
- 中国・日本・インド以外の各国も、NGDPDはわずかに上昇傾向であり、PPPSHはわずかに減少傾向かほぼ横ばいである

※1 NGDPD: 市場の為替レート（年平均）を使用して米ドルに換算された自国通貨の現在価格の国内総生産(GDP)
※2 PPPSH: 世界全体の購買力平価（PPP）シェアに基づく国内総生産
※3 アメリカの数値は今回使用したデータに含まれていないため、実際の順位は異なる可能性もある

### 東アジア
以下の各種結果からわかることは、
- 日本は
    - 2004年〜2017年にかけて人口が僅かに減少していて、労働所得も減少しているが、中国よりは労働所得は高い
    - 2012年まで人口増加とともに輸入額も増加したが、そこから人口減少に伴って2020年にかけて輸入額も減少している
    - 中国と比較すると、人口が約10分の1に対して輸入額が約3分の1となっており、日本の方がより輸入に依存している
    - 2011年まで人口増加とともに輸出額も年々増加したが、そこから人口減少に伴って2020年にかけて輸入額も減少している
    - 2012年まで人口増加とともにNGDPDも年々増加し、2015年にかけて人口減少とともにNGDPDも減少したが、2026年にかけてNGDPDが増加している
    - 1991年まで人口増加とともにPPPSHも年々増加したが、2008年までは人口は増加してもPPPSHは減少し、さらに2026年にかけて人口減少とともにPPPSHも減少している
- 中国は
    - 2010年〜2015年にかけて急激に労働所得が年々増加したが、2015年〜2017年にかけては減少している
    - 2018年まで人口増加とともに輸入額も年々増加したが、2020年にかけてわずかに減少している
    - 2020年まで人口増加とともに輸出額も年々増加している
    - 2026年まで人口増加とともにNGDPD・PPPSHも年々増加している
    - 日本と比較すると、労働所得は低いものの、<font color="Red">輸出入額は2倍以上、NGDPDは3倍以上、PPPSHは2倍以上である</font>
